# 🚀 Premium Software Engineering Portfolio

A modern, highly polished, visually memorable, and fully responsive software engineering portfolio built for **Prashant Srivastava** using **React, TypeScript, Tailwind CSS (v4), and Vite**.

## 🎨 Visual Design System
- **Theme**: Minimalist, art-directed editorial layout featuring warm off-white, soft lavender, powder blue, subtle peach, muted indigo, and dark navy typography.
- **Components**: Bento grids, asymmetric layouts, custom technical illustrations, and responsive media query scaling.
- **Recruiter Focus**: Instant access to resume download, clean navigation, key project metrics, and problem-solving statistics (200+ solved DSA challenges).

## 📁 Key Projects Featured
1. **Mini Search Engine**: Java implementation of an inverted index, TF-IDF ranking, and cosine similarity vector retrieval.
2. **Resume Analyzer**: Python natural language parser and skill gap analysis comparison engine.
3. **Consumer Behavior Analysis**: Python data analysis (NumPy/Pandas) and interactive Power BI dashboards.
4. **Team Task Manager**: Object-Oriented clean architecture design in Java supporting priority queue scheduling.

## 🛠️ Technical Stack
- **Core**: React 19, TypeScript
- **Styling**: Tailwind CSS v4, Custom CSS (Off-white & Pastel Theme)
- **Icons**: Lucide React
- **Animations**: Motion (Reduced-motion compliant)
- **Build Tool**: Vite

## 🚀 Development & Local Setup

### Prerequisites
- Node.js (v18+)

### Steps
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
